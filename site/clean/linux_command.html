Linux Commandline Tips & Tricks

the linux commandline the most efficient wat to work with linux - mastering this feature will speed-up the work segnificantly. I will attemt to record some of the methods I use in my day to day work

command return value

each command has a return value. the command value in the code can usually be by the 'exit ' command.
a command succesecc will return '0' value and a failuer will return '1' or any non-zero value depending on the running software
one can check the return value by the system variable '?' - by echoing the variable with 'echo $?' one can see the return value of the last command run
for example: test@test:~ $ ps
PID TTY TIME CMD
20748 pts/34 00:00:00 bash
20846 pts/34 00:00:00 ps
test@test:~ $:~ $ ps | grep bush
test@test:~ $:~ $ echo $?
1
test@test:~ $:~ $ ps | grep bash
20748 pts/34 00:00:00 bash
test@test:~ $:~ $ echo $?
0
 as seen in the example if 'grep' was able to find a match it will return '0', if not it will return '1'
