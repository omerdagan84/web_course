parsing and validating IP addresses

#!/bin/bash
function valid_ip()
{
	local  ip=$1
	local  stat=1

	if [[ $ip =~ ^[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}\.[0-9]{1,3}$ ]]; then
		OIFS=$IFS
		IFS='.'
		ip=($ip)
		IFS=$OIFS
		[[ ${ip[0]} -le 255 && ${ip[1]} -le 255 \
			&& ${ip[2]} -le 255 && ${ip[3]} -le 255 ]]
		stat=$?
	fi
	return $stat
}

if [ -z $1 ]; then 
	echo " no IP given as parameter - reverting to default"
	IP=172.0.0.0
else
	IP=$1
fi
if valid_ip $IP; then 
	echo "entered IP is valid"
else
	echo "entered IP is invalid"
	exit
fi

echo "at this point you can add your commands using IP variable"
I usually use many scripts to work with my embedded devices - mainly using ssh and scp commands
since these use IP addresses I use this script template. It accepts the IP in the first parameter and validates that it is in the IPv4 format i.e. [0-255].[0-255].[0-255].[0-255]
It also contains a default IP to use if none is given.
