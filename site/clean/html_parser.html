Simple HTML parser

Intro

well, since I started working on this page, I noticed that writing HTML tags pisses me off
It's annoying. Since I don't like to be annoyed I figured I will write a small script to ease writing HTML code.

obviously, I am not going to write a huge program to actually handle the entire language restrictions. there are enough IDE to handle writing and managing HTML code.
No, I just want to be able to write the textual parts of the page easily, so I don't need to hang on writing all the tags when writing.

the script

definitions

As a starting point I decided to support easy writing of just a few textual tags
i.e. p, h1, h2, h3, ul and li.
The idea is being able to write naturally as possible and letting the script parse the text file and create the needed tags
I decided on the most easy to implement method, a simple line based parsing where each line will be:
PARAM text content
where PARAM is
  • p - a start paragraph definition
  • h1 - a header-1 definition
  • h2 - a header-2 definition
  • h3 - a header-3 definition
  • ul - an unordered list definition
  • li - a list item

design restrictions

the script should be able to support multi-line paragraphs
the writing should remain as natural as possible
