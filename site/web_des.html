Web design

In order to obtain the basic knowledge I deem needed to fulfil my goals i used a lot of resources.
Due mind, I know that there are a lot of tools to shorten web development, as well as doing this up-to standard.
a lot of the things that i do are considered dirty or bad coding. I agree but the thing is at least for me is learning and all of these automated tools usually just hide the actual implementation from the developer.
so i took another approach - do it manually slowly - so i can actually learn the basics.
as with everything else once you worked yourself through the basics, everything else seems easier.
I will try and document some to give anyone interested a starting point:

SoloLearn

a very nice start to learn some coding; for me what did it was the App - you can learn from anywhere on your cellphone or tablet.
now, do mind that coding on any portable device, at least for me, is not really helpful. but it did enable me to utilize my commute time; I used it to learn the basics and at home i finished the rest

W3 schools

I doubt that anyone that wants to learn anything related to the web has not encountered this site a some point.
It is extensive, sometimes it might be too extensive at least if you are coming to the process without any prior experience.
I still use it as reference - at least for that purpose it is unmatched

CSS grid

CSS grid - i found this to be extremely helpful tool.
I admit CSS styling with direct box\div control was extremely difficult to me. This allows one to design nice looking pages very easily
